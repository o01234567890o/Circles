# Circles 画圆圈⚪
 
在窗口内生成一些随机大小的圆形，颜色也是随机的。当鼠标进入圆形的范围内时，圆圈会放大，距离圆心越近越大，最大放大到之前半径的3倍，鼠标离开后又会恢复原样。改变窗口大小或者点击鼠标右键会重新生成。
